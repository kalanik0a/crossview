The silo

~3,700 canonical MITRE entities (CWE/CAPEC/ATT&CK/ATLAS/D3FEND/UKC) in one offline, cross-referenced graph. Query by ID, full-text search, or GraphQL.

The scanner

A 5-stage pipeline — survey → prematch → investigate → verify → report — that confirms reachability and walks each finding through the graph.

Real-world signal

NVD CVEs intersected with CISA KEV, so "this is a bug" becomes "this weakness class is exploited in the wild, 9 in ransomware."

Intel fusion

LLM-generated, grounded threat-intel reports attach to the same entity nodes — findings, taxonomy, exploitation data, and narrative intel share one index.

Security posture

Double-audited against 32 CWE categories (all PASS). 5 automated SAST scanners clean (Bandit, Semgrep, detect-secrets, Gitleaks, Trivy). 26 findings found and fixed across 2 passes.

Output formats

Markdown · SARIF 2.1.0 · STIX 2.1 · HTML — integrates with GitHub Code Scanning, VS Code, and threat intel platforms out of the box.

Enrichment

CISA Known Exploited Vulnerabilities (KEV) · NVD CVE bulk sweep · web research via crawl4ai — all cached in local SQLite, resumable, TTL-managed.

Agent-ready

Claude Code skill, MCP-compatible. Any AI agent can invoke Crossview to scan a codebase and receive structured SARIF findings. Pairs with Touchstone for hardware-bound consent.

SARIF · STIX · GraphQL · TUI — self-hosted, zero external dependencies for the core.
